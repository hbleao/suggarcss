# Ocean Design System

Bem-vindo à documentação do Ocean Design System, a biblioteca de componentes React da Porto Seguro.

## Visão Geral

O Ocean Design System é uma biblioteca de componentes React com SASS que implementa a linguagem visual da Porto Seguro. Foi projetada para criar experiências digitais consistentes, acessíveis e de alta qualidade em todos os produtos e serviços da Porto.

## Principais Recursos

- **Componentes React**: Uma coleção completa de componentes React prontos para uso
- **Design Responsivo**: Todos os componentes são totalmente responsivos e adaptáveis a diferentes tamanhos de tela
- **Acessibilidade**: Componentes desenvolvidos seguindo as diretrizes WCAG para garantir acessibilidade
- **Temas e Variantes**: Suporte a diferentes temas e variantes para os diversos negócios da Porto
- **Tipografia Consistente**: Sistema tipográfico padronizado para manter a consistência visual
- **Sistema de Grid**: Sistema de grid flexível e responsivo para layouts consistentes
- **Hooks Personalizados**: Coleção de hooks React para funcionalidades comuns

## Instalação

```bash
npm install @porto/js-library-corp-hubv-porto-ocean
```

ou

```bash
yarn add @porto/js-library-corp-hubv-porto-ocean
```

## Uso Básico

```jsx
import { Button, Card, Header } from '@porto/js-library-corp-hubv-porto-ocean';

function App() {
  return (
    <div>
      <Header variant="insurance" />
      <Card>
        <h2>Bem-vindo ao Ocean Design System</h2>
        <p>Comece a construir interfaces incríveis!</p>
        <Button variant="primary">Começar</Button>
      </Card>
    </div>
  );
}
```
