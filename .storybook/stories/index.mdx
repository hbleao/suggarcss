# Sistema de Design Ocean

![Logo Ocean](https://via.placeholder.com/200x60?text=Ocean)

O Ocean é uma biblioteca de componentes React em TypeScript que fornece um conjunto de componentes reutilizáveis, estilizados e acessíveis para construir interfaces modernas e consistentes.

## Instalação

Para instalar o Ocean em seu projeto, use npm ou yarn:

```bash
# Usando npm
npm install @porto/js-library-corp-hubv-porto-ocean

# Usando yarn
yarn add @porto/js-library-corp-hubv-porto-ocean
```

## Uso Básico

```jsx
import React from 'react';
import { Button, Typography } from '@porto/js-library-corp-hubv-porto-ocean';
import '@porto/js-library-corp-hubv-porto-ocean/dist/styles.css';

function App() {
  return (
    <div>
      <Typography variant="title1">Olá, Ocean!</Typography>
      <Button variant="insurance" styles="primary">Clique em mim</Button>
    </div>
  );
}

export default App;
```

## Exemplo com Hooks

```jsx
import React from 'react';
import { Modal, Button } from '@porto/js-library-corp-hubv-porto-ocean';
import { useModal } from '@porto/js-library-corp-hubv-porto-ocean/hooks';

function ExemploModal() {
  const { isOpen, openModal, closeModal } = useModal();
  
  return (
    <>
      <Button onClick={openModal}>Abrir Modal</Button>
      
      <Modal 
        title="Exemplo de Modal"
        subtitle="Usando o hook useModal para controle de estado"
        isOpen={isOpen}
        onClose={closeModal}
      >
        <p>Conteúdo do modal aqui.</p>
        <Button onClick={closeModal}>Fechar</Button>
      </Modal>
    </>
  );
}

export default ExemploModal;
```
