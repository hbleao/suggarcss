## O que são tokens de cores?

Tokens de cores são variáveis que representam valores de cores específicos em um sistema de design. Eles permitem manter consistência visual em toda a aplicação e facilitam alterações globais de cores quando necessário.

## Uso das Cores

As cores do Ocean Design System são usadas para comunicar significado e criar hierarquia visual:

- **Cores da Marca**: Usadas para elementos de marca e identidade visual
- **Cores Neutras**: Usadas para texto, fundos e elementos estruturais
- **Cores de Acento**: Usadas para destacar elementos importantes, alertas e feedback

### Como usar os tokens de cores

Em SCSS:
```scss
.my-element {
  color: $color-brand-insurance-900;
  background-color: $color-neutral-100;
}
```

Em CSS:
```css
.my-element {
  color: var(--ocean-color-brand-insurance-900);
  background-color: var(--ocean-color-neutral-100);
}
```

Em componentes React:
```jsx
<Button variant="insurance" styles="primary">Botão Primário</Button>
```
