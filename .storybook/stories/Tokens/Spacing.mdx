# Espaçamento

O Ocean Design System utiliza um sistema de espaçamento consistente para criar layouts harmoniosos e manter o ritmo visual em toda a interface.

## O que são tokens de espaçamento?

Tokens de espaçamento são variáveis predefinidas que representam distâncias específicas usadas para margens, preenchimentos e gaps entre elementos. Eles garantem consistência visual e facilitam a manutenção do design.

## Uso dos Tokens de Espaçamento

Os tokens de espaçamento devem ser usados para todas as propriedades relacionadas a espaço, como:

- Margens (`margin`, `margin-top`, `margin-right`, etc.)
- Preenchimentos (`padding`, `padding-top`, `padding-right`, etc.)
- Espaçamento entre elementos (`gap`)
- Posicionamento de elementos

### Como usar os tokens de espaçamento

Em SCSS:
```scss
.my-element {
  margin-bottom: $space-md;
  padding: $space-sm $space-lg;
}

.grid-container {
  gap: $space-xl;
}
```

Em CSS:
```css
.my-element {
  margin-bottom: var(--ocean-space-md);
  padding: var(--ocean-space-sm) var(--ocean-space-lg);
}

.grid-container {
  gap: var(--ocean-space-xl);
}
```

## Boas Práticas

- Use sempre tokens de espaçamento em vez de valores arbitrários
- Mantenha a consistência usando os mesmos tokens para elementos similares
- Use espaçamentos maiores para separar seções e menores para elementos relacionados
- Considere a hierarquia visual ao aplicar espaçamentos (elementos mais importantes podem ter mais espaço ao redor)
- Adapte o espaçamento para diferentes breakpoints quando necessário
