# Loader

O componente `Loader` é usado para indicar ao usuário que uma operação está em andamento, fornecendo feedback visual durante processos de carregamento, envio de dados ou qualquer operação assíncrona.

## Props

| Prop | Tipo | Padrão | Descrição |
|------|------|--------|-----------|
| `color` | `string` | `'neutral-900'` | Cor do loader, usando tokens de cor do sistema |
| `size` | `number` | `24` | Tamanho do loader em pixels |
| `className` | `string` | - | Classes CSS adicionais para o loader |
| `style` | `CSSProperties` | - | Estilos inline adicionais para o loader |



## Boas Práticas

- Use o Loader para operações que levam mais de 300ms para concluir
- Para operações muito rápidas, considere não mostrar um loader para evitar flashs na interface
- Combine com mensagens descritivas para informar o usuário sobre o que está sendo carregado
- Para operações longas, considere usar um loader com progresso determinado
- Use o tamanho apropriado para o contexto - loaders menores para componentes individuais e maiores para seções inteiras
- Quando usar overlay, certifique-se de que o contraste entre o loader e o fundo é adequado
- Em formulários, desative os botões de envio durante o carregamento para evitar envios duplicados
