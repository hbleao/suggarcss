# Progress

O componente `Progress` é usado para mostrar o progresso de uma operação ou indicar o status de conclusão de uma tarefa, fornecendo feedback visual ao usuário.



## Importação

```jsx
import { Progress } from '@porto/js-library-corp-hubv-porto-ocean';
```

## Uso Básico

```jsx
import React from 'react';
import { Progress, Typography } from '@porto/js-library-corp-hubv-porto-ocean';

function Example() {
  return (
    <div style={{ display: 'flex', flexDirection: 'column', gap: '24px' }}>
      <div>
        <Typography variant="body2" style={{ marginBottom: '8px' }}>
          Progresso: 30%
        </Typography>
        <Progress value={30} />
      </div>
      
      <div>
        <Typography variant="body2" style={{ marginBottom: '8px' }}>
          Progresso: 70%
        </Typography>
        <Progress value={70} />
      </div>
      
      <div>
        <Typography variant="body2" style={{ marginBottom: '8px' }}>
          Progresso: 100%
        </Typography>
        <Progress value={100} />
      </div>
    </div>
  );
}
```

## Props

| Prop | Tipo | Padrão | Descrição |
|------|------|--------|-----------|
| `value` | `number` | `0` | Valor atual do progresso (0-100) |
| `max` | `number` | `100` | Valor máximo do progresso |
| `variant` | `'default' \| 'success' \| 'warning' \| 'error' \| 'info'` | `'default'` | Variante de cor do progresso |
| `size` | `'small' \| 'medium' \| 'large'` | `'medium'` | Tamanho da barra de progresso |
| `type` | `'line' \| 'circle'` | `'line'` | Tipo de indicador de progresso |
| `showInfo` | `boolean` | `false` | Se verdadeiro, exibe o valor percentual |
| `format` | `(percent: number) => ReactNode` | - | Função para formatar o texto de informação |
| `strokeWidth` | `number` | `8` | Largura da linha de progresso |
| `strokeLinecap` | `'round' \| 'square'` | `'round'` | Estilo das extremidades da linha |
| `indeterminate` | `boolean` | `false` | Se verdadeiro, exibe um progresso indeterminado (animação) |
| `className` | `string` | - | Classe CSS adicional para o componente |
| `style` | `CSSProperties` | - | Estilos inline adicionais |



## Boas Práticas

- Use o componente `Progress` para operações que levam mais de um segundo para completar
- Escolha a variante de cor apropriada para o contexto (ex: verde para sucesso, vermelho para erro)
- Para operações de duração desconhecida, use o modo indeterminado
- Combine o componente com mensagens de texto descritivas para explicar o que está acontecendo
- Considere mostrar o valor percentual para operações que levam mais tempo
- Em uploads de arquivos, mostre o progresso individual para cada arquivo
- Para operações complexas com várias etapas, considere usar um progresso por etapas
- Evite mudanças bruscas no valor do progresso, prefira transições suaves
