# BannerHero

O componente BannerHero é usado para criar seções de destaque em páginas, geralmente no topo, apresentando conteúdo importante com impacto visual. Ideal para landing pages, páginas iniciais e seções de destaque.

## Importação

```jsx
import { BannerHero } from '@porto/js-library-corp-hubv-porto-ocean';
```

## Uso Básico

```jsx
<BannerHero
  title="Título Principal"
  subtitle="Subtítulo ou descrição complementar"
  imageUrl="/images/banner-background.jpg"
  ctaLabel="Saiba Mais"
  ctaUrl="/detalhes"
/>
```

## Props

| Prop | Tipo | Padrão | Descrição |
|------|------|--------|-----------|
| `bgColor` | `ColorToken` | `'neutral-0'` | Cor de fundo do banner |
| `title` | `ReactNode` | - | Título principal do banner |
| `subtitle` | `ReactNode` | - | Subtítulo do banner |
| `text` | `ReactNode` | - | Texto descritivo do banner |
| `logo` | `ReactNode` | - | Logo a ser exibido no topo do conteúdo |
| `image` | `ReactNode` | - | Imagem principal do banner |
| `buttons` | `BannerHeroButton[]` | `[]` | Array de botões de ação |
| `stores` | `BannerHeroStore[]` | `[]` | Array de links para lojas de aplicativos |
| `contentProps` | `RowProps` | - | Props adicionais para o componente Row que envolve o conteúdo |

## Variações

### Variantes Visuais

O BannerHero suporta diferentes variantes visuais para se adequar ao contexto da página:

```jsx
// Variante primária (padrão)
<BannerHero
  variant="primary"
  title="Variante Primária"
  subtitle="Ideal para destaques principais"
  imageUrl="/images/primary-banner.jpg"
  ctaLabel="Começar Agora"
  ctaUrl="/start"
/>

// Variante secundária
<BannerHero
  variant="secondary"
  title="Variante Secundária"
  subtitle="Para seções complementares"
  imageUrl="/images/secondary-banner.jpg"
  ctaLabel="Descobrir"
  ctaUrl="/discover"
/>

// Variante escura
<BannerHero
  variant="dark"
  title="Variante Escura"
  subtitle="Para um visual mais dramático"
  imageUrl="/images/dark-banner.jpg"
  ctaLabel="Explorar"
  ctaUrl="/explore"
/>
```

## Personalização Avançada

### Com Conteúdo Personalizado

Você pode usar a prop `children` para adicionar conteúdo personalizado ao banner:

```jsx
<BannerHero
  title="Banner com Conteúdo Personalizado"
  imageUrl="/images/banner.jpg"
>
  <div style={{ marginTop: '24px', backgroundColor: 'rgba(255, 255, 255, 0.8)', padding: '16px', borderRadius: '8px' }}>
    <h3>Conteúdo Personalizado</h3>
    <p>Você pode adicionar qualquer conteúdo React aqui para personalizar seu banner.</p>
    <div style={{ display: 'flex', gap: '8px', marginTop: '16px' }}>
      <Button variant="primary">Botão Principal</Button>
      <Button variant="secondary">Botão Secundário</Button>
    </div>
  </div>
</BannerHero>
```

### Com Sobreposição Personalizada

Você pode ajustar a opacidade da sobreposição para controlar o contraste entre o conteúdo e a imagem de fundo:

```jsx
// Sobreposição mais clara
<BannerHero
  title="Sobreposição Sutil"
  subtitle="A imagem de fundo é mais visível"
  imageUrl="/images/banner.jpg"
  overlayOpacity={0.2}
/>

// Sobreposição padrão
<BannerHero
  title="Sobreposição Padrão"
  subtitle="Equilíbrio entre imagem e legibilidade"
  imageUrl="/images/banner.jpg"
  overlayOpacity={0.4}
/>

// Sobreposição mais escura
<BannerHero
  title="Sobreposição Intensa"
  subtitle="Para maior legibilidade do texto"
  imageUrl="/images/banner.jpg"
  overlayOpacity={0.7}
/>
```

## Acessibilidade

O componente BannerHero implementa as melhores práticas de acessibilidade:

- Usa estrutura semântica HTML apropriada
- Garante contraste adequado entre texto e fundo
- Fornece texto alternativo para imagens
- Mantém todos os elementos interativos acessíveis por teclado

## Responsividade

O BannerHero é totalmente responsivo:

- Em dispositivos móveis, o tamanho e o espaçamento são ajustados automaticamente
- O texto é redimensionado para manter a legibilidade em diferentes tamanhos de tela
- A imagem de fundo é ajustada para diferentes proporções de tela

## Boas Práticas

- Use imagens de alta qualidade com boa resolução
- Mantenha o texto conciso e impactante
- Certifique-se de que há contraste suficiente entre o texto e a imagem de fundo
- Escolha imagens que complementem o conteúdo e a mensagem
- Evite sobrecarregar o banner com muitos elementos ou texto
- Considere o contexto da página ao escolher o tamanho e a variante do banner
