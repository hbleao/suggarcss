# Arredondamento (Border Radius)

O Ocean Design System utiliza um sistema consistente de arredondamento de bordas para criar uma linguagem visual coesa e moderna.

## O que são tokens de arredondamento?

Tokens de arredondamento (border radius) são variáveis predefinidas que representam o grau de curvatura aplicado aos cantos de elementos da interface. Eles ajudam a manter a consistência visual e definem parte da personalidade do design system.

## Escala de Arredondamento

O Ocean oferece uma escala de arredondamento que vai de nenhum arredondamento até o arredondamento completo (circular):

## Uso dos Tokens de Arredondamento

Os tokens de arredondamento devem ser aplicados de forma consistente em componentes similares para manter a coerência visual da interface.

### Aplicação Comum

- **Botões**: Geralmente usam $radius-md (4px)
- **Cards**: Podem usar $radius-lg (8px) ou $radius-xl (12px)
- **Modais**: Geralmente usam $radius-xl (12px) ou $radius-2xl (16px)
- **Chips/Tags**: Podem usar $radius-full para um visual mais arredondado
- **Campos de formulário**: Geralmente usam $radius-md (4px)

### Como usar os tokens de arredondamento

Em SCSS:
```scss
.button {
  border-radius: $radius-md;
}

.card {
  border-radius: $radius-lg;
}

.avatar {
  border-radius: $radius-full;
}
```

Em CSS:
```css
.button {
  border-radius: var(--ocean-radius-md);
}

.card {
  border-radius: var(--ocean-radius-lg);
}

.avatar {
  border-radius: var(--ocean-radius-full);
}
```

## Boas Práticas

- Mantenha a consistência usando o mesmo arredondamento para componentes similares
- Use arredondamentos maiores para elementos que deseja destacar ou tornar mais amigáveis
- Use arredondamentos menores ou nenhum arredondamento para interfaces mais sérias ou técnicas
- Considere a personalidade da marca ao escolher o grau de arredondamento
- Lembre-se que o arredondamento pode afetar a percepção de tamanho dos elementos
