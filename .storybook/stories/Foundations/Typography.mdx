# Tipografia

O Ocean Design System utiliza um sistema tipográfico cuidadosamente projetado para garantir legibilidade, hierarquia e consistência em todas as interfaces.

## O que são tokens de tipografia?

Tokens de tipografia são variáveis que definem as características visuais do texto em um sistema de design, como família de fonte, tamanho, peso e estilo. Eles garantem consistência tipográfica em toda a aplicação e facilitam a manutenção do design.

## Componente Typography

O componente Typography é a maneira recomendada de aplicar estilos tipográficos consistentes em toda a aplicação.

### Exemplo de Uso

```jsx
// Título principal
<Typography variant="title1" as="h1" color="neutral-900" weight="bold">
  Bem-vindo ao portal
</Typography>

// Texto de apoio com cor personalizada
<Typography variant="body2" color="neutral-500">
  Preencha os campos obrigatórios
</Typography>

// Componente com estilo itálico
<Typography variant="body1" fontStyle="italic" as="span">
  Este é um texto em itálico.
</Typography>
```

### Propriedades do Componente

| Propriedade | Tipo | Padrão | Descrição |
|-------------|------|--------|-----------|
| variant | string | 'title1' | Define o estilo visual da tipografia ('title1', 'title2', 'body1', etc.) |
| as | string | 'h1' | Define o elemento HTML a ser renderizado ('h1', 'p', 'span', etc.) |
| color | string | 'neutral-900' | Define a cor do texto (usando os tokens de cor) |
| weight | string | 'regular' | Define o peso da fonte ('light', 'regular', 'medium', 'semibold', 'bold') |
| fontStyle | string | 'normal' | Define o estilo da fonte ('normal', 'italic') |
| className | string | '' | Permite adicionar classes CSS personalizadas |

## Boas Práticas

- Use as variantes tipográficas pré-definidas sempre que possível para manter a consistência
- Limite o número de tamanhos e pesos de fonte em uma única tela para evitar poluição visual
- Mantenha um bom contraste entre o texto e o fundo para garantir a acessibilidade
- Use pesos de fonte para criar hierarquia, não apenas tamanhos diferentes
- Respeite a hierarquia tipográfica: títulos maiores para conteúdo mais importante, textos menores para conteúdo secundário
