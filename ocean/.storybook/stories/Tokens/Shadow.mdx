## O que são tokens de sombra?

Tokens de sombras são variáveis predefinidas que representam diferentes níveis de elevação através de sombras projetadas. Eles ajudam a criar uma sensação de profundidade e hierarquia na interface, indicando visualmente a relação entre os elementos.

## O que são tokens de sombra?

## Uso dos Tokens de Sombra

As sombras devem ser aplicadas de forma consistente para criar uma hierarquia visual clara na interface.

### Aplicação Comum

- **$shadow-xs e $shadow-sm**: Para elementos sutilmente elevados como cartões, botões em estado normal
- **$shadow-md**: Para elementos em hover, componentes interativos
- **$shadow-lg**: Para elementos em foco, dropdowns, popovers
- **$shadow-xl e $shadow-2xl**: Para modais, diálogos, elementos flutuantes
- **$shadow-3xl**: Para elementos que precisam de máximo destaque

### Como usar os tokens de sombra

Em SCSS:
```scss
.card {
  box-shadow: $shadow-sm;

  &:hover {
    box-shadow: $shadow-md;
  }
}

.modal {
  box-shadow: $shadow-xl;
}
```

Em CSS:
```css
.card {
  box-shadow: var(--ocean-shadow-sm);
}

.card:hover {
  box-shadow: var(--ocean-shadow-md);
}

.modal {
  box-shadow: var(--ocean-shadow-xl);
}
```

## Boas Práticas

- Use sombras para indicar hierarquia e interatividade
- Evite usar sombras muito intensas em elementos pequenos
- Combine sombras com transições suaves para criar interações fluidas
- Mantenha consistência no uso das sombras para elementos com funções similares
- Considere o contexto de uso: interfaces mais leves podem usar sombras mais sutis, enquanto interfaces mais imersivas podem usar sombras mais pronunciadas
